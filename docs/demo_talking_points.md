# Talking Points

## Mo dau

"Ban dieu hanh muon biet trong hang chuc nghin phan hoi thang nay, co bao nhieu phan hoi thuc chat dang noi ve cung mot van de nghiem trong: giao dich that bai nhung tai khoan van bi tru tien. Neu tim bang keyword nhu `tru tien`, `mat tien`, `ghi no`, `hoan tien`, chung ta se bo sot rat nhieu cach dien dat khac nhau."

## Man 1: keyword search bi sot

Chay query `A2` trong `sql/06_demo_queries.sql`.

Noi ngan gon: "Cach nay phu thuoc vao tu khoa. Khach hang viet `so du giam`, `ghi no`, `tien bi giu` thi query keyword co the khong bat duoc."

## Man 2: semantic search

Chay query `B`.

Diem nhan: SQL Server tra ve cac phan hoi cung ban chat, du khong trung tu khoa:

- "Thanh toan loi nhung so du trong vi van giam."
- "Don hang khong thanh cong, tien chua duoc hoan."
- "Giao dich bi treo sau khi xac thuc, tai khoan da ghi no."
- "App bao timeout nhung tien trong tai khoan bi giu."

## Man 5: bao mat

Chay query `F`.

"Native Vector Search giai quyet phan luu tru, indexing va truy van vector trong SQL Server. De dam bao du lieu khong roi firewall, embedding model duoc cau hinh local/private. Phan hoi khach hang duoc bien thanh vector trong vung kiem soat cua doanh nghiep."

## Cau chot

"Voi SQL Server 2025 Native Vector Search, chung ta luu embedding, tao index va truy van ngu nghia ngay trong he quan tri du lieu quen thuoc. Dieu nay giam rui ro ro ri du lieu, giam do phuc tap kien truc va rut ngan thoi gian phat hien van de khach hang."
